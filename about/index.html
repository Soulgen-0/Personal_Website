How I Started
My Programming Journey

I started my programming journey on a multiplayer online platform known as Roblox. The language used on Roblox is Luau, which is based on Lua. I learned all the basics of programming from there, such as functions, loops, variables, scopes and learned more advanced things like modular programming, OOP and more.

After trying to make a game on Roblox to no avail (I struggled with perfectionism), I took a break for a couple of months before I realized that my passion doesn't just live in making games.. It also exists in building! After realizing that, I decided to learn Python for awhile and made some Discord bots and CLI programs.

After making a number of python projects, I decided I also wanted to learn Web Dev.. Which meant it was time to learn HTML, CSS and JavaScript! I am still learning these languages but I hope to become proficient enough to make some amazing websites and apps!
